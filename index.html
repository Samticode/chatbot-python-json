Follow these steps:

  1. First you need to know the URL address of the radio station you would like to add. Note that the URL address needs to be exactly the streaming link, like \u2018http://http-live.sr.se/p1-mp3-192\u2019 in the example below. Otherwise, it won\u2019t be played when you select it in nordicComfortPlayer.
  1. Go to C:\\Users\\AppData\\Local\\ComfortPlayer and open \u2018settigs.json\u2019 file with Notepad. Scroll down and find the section where all the radio stations are located. The section starts with the syntax \"radio\".  
  1. Under this section you can add you desired radio station. Below is a syntax example that shows how the URL should be added. In front of \"title\", you can give a name to the new radio station, which will then appear in the media library of nordicComfortPlayer.

       {

           \"title\": \"Sverige Radio P1\",

           \"url\": \"http://http-live.sr.se/p1-mp3-192\"

       },

  1. Save the changes and start nordicComfortPlayer. You should be able to see and play the new radio station.

There are two ways to change the orientation of the image:

  • If you have nordicComfortPlayer, open the app and select one of the icons on the top right according to whether the screen is at the patient\u2019s head or feet, and whether you have a single or double mirror.
  • If you do not have the nordicComfortPlayer app, there are flip/rotate controls on the LCD as well. You can use the touch buttons on the front top left corner of the LCD to adjust the orientation (see the table below for the functionality of the touch buttons). Activate the on-screen menu and then navigate to and enter the \u2018other\u2019 submenu. Then enter the \u2018Display rotation\u2019 option and choose one of the four orientation options (normal, H, V, H/V). The selected setting is stored and will be retained after power cycling.

\u200d

In both approaches, always check with patients if they can see everything in the correct orientation.
